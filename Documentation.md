# API doc

## Todos API

Endpoint: /api/todos

* GET: Read all posts of logged in user (requires auth)
* POST: Update post (requires auth)
* PUT: Create new post (requires auth)
* DELETE: Remove post (requires auth)

## Users API

Endpoint: /api/users

* GET: Read logged in user (requires auth)
* POST: Try to log in
* PUT: Create new User
* DELETE: Remove currently logged in user (requires auth)
